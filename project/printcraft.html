What is 3D Printing?

  • Materials: PLA, PETG, ABS, Nylon, Metal, and more!

  • Processes: Bring your ideas to life layer by layer from a digital design.

  • Types: FDM, SLA, SLS, DMLS, and more!

  • Use-Cases: Prototyping, prosthetics, hobbies, and wherever else your imagination takes you.

How 3D printing works

Design

Create or download a 3D model using CAD software or a model library.

Slice

Slicing software converts the model into layers and generates the instructions for the printer.

Print

The printer builds the object layer-by-layer using the materials provided.
